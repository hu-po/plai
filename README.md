## plai

![Plai in a scifi landscape](assets/coverimage.png)

Plai is an robotic cat toy that uses LLMs to entertain your cat. Some of the technologies used:

- Raspberry Pi
- GPT-4 API
- DynamixelSDK
- OpenCV
- PyTorch
